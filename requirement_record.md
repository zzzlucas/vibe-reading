# 系统核心机制与复杂需求记录

## 1. 核心阅读与装饰风格系统
- **动态风格分级系统**：涵盖 VS Code, Terminal, IDEA, 掘金, Stack Overflow 等皮肤。Gemini、MDN、ChatGPT 为免费版，其余归属于 Pro 系列。
- **“经典博客1”布局风格 (classic_blog1)**：
  - **高度还原引擎**：完全替换标准的 Sidebar 结构，利用 `ClassicBlog1Layout.vue` 提供定制化顶部蓝条导航、二级统计数据、自带博主信息与侧边原生博客链接栏位。
  - **博客翻页交互 (Pagination)**：为主界面增加了内嵌 Scrollbar 与大量作品条目下的翻页支持；主内容区取消极端的 `max-width` 限制实现全宽自适应平铺。
- **全局项目混淆与下沉模式**：通过 `betaText` 自定义限免标签。允许左上角项目标题随时进行更名替代，并且预留双击此文字快速激活“BOSS 键”的功能通道。非AI聊天皮肤下的界面，左下角的“设置”按钮以收缩极简形态悬浮存在，降低视觉违和。
- **“作品”内容级二次排版 (规划/实现中)**：引入独立于正文 Obfuscation 的“功能内容块”（如鉴权存根、开源证书），成对包裹打乱小说自身文本结构。

## 2. BOSS 机制与场景防御系统 (Boss Key Settings)
- **多层级逃生触发槽 (Multi-level Escape Triggers)**：默认仅支持 **反引号 \`**（Esc键下方）作为键盘触发键，原有的 `Escape` 已从默认列表中移出。**Escape 键现已统一并锁定为系统级操作**：专门用于快速关闭所有弹窗、设置页及帮助面板，不再参与老板键触发。此外，保留了隐秘盲操作的**“鼠标左键连击三下”**强制逃脱模式。
- **AI 动态数据流伪造策略**：
  - 支持后台动态拉取预置的假数据文案池(IT/设计/职场，**默认模板已更新为进阶 Excel 函数技巧**)或复用指定的 AI 历史对话充当防备屏幕流。
  - **精确流控**：流式生成可配置定位于当前内容的 First Turn 或 Last Turn，同时能够触发底层自动滚页。
  - **防串联重绘阻挡**：进入 Boss 模式后流式动画具备单次触发防护，不会在频繁点击切换菜单时发生多余且生硬的表现重置。
  - 根据数据对象 `type` 字段分类（`novel` | `dummy` | `ai`），实时重排与挂载隐蔽的假侧边栏。默认流输出字符量定义为 150 词。

## 3. 面向大模型直连与 AI 行为渲染 (LLM Integration & Chat flow)
- **本地直连引擎**：加密处理并将用户配置的 `API Key`、`Base URL` 存储在前端本地容器。架构内独立维护 4 个预设配装（常规交流/深度思考/纯视觉模态/绘图引擎）用以随时按纽切换。
- **多会话并发解耦合**：从底层将不同对话标签页的上下文容器拆离。支持 A 对话正在长时运算输出 (CoT推导) 的同时，用户在 B 对话内畅所欲言，互相维护各自特定的中断句柄 (`AbortController`) 和内存渲染树。
- **气泡与细节渲染特化 (Refine Message Displays)**：
  - 打通 TTFT 波浪点骨骼动画。引入 `<details>` 完美支持包含 DeepSeek-R1 级别返回带有的过程推导区块折叠互动。并规避常规首行缩进影响 Markdown 引擎列表标签的解析崩塌。
  - **用户气泡智能重构**：完全解开用户气泡与“作品标题”同步显示的错误强关联绑定。AI 主题中永远可见；博客类主题取决于内容的 `userTextHtml` 字段；阅读型文档则将其透明摘除。

## 4. 数据仓储持久层与本地化操作验证
- **本地无界限存储判定 (Pro storage)**：
  - 利用系统浏览器的 IndexedDB 或兼容 Cookie 执行设备级指纹的加密认证分发。用来闭环解决诸如 Pro 版“隐藏状态”验证等多级鉴权逻辑，杜绝了服务器端的风险侵入。
  - 对于超出 30 万字额度的限制卡控，依赖原生静默比对策略提醒阻断。提供物理级“总体积量化管控中心”，实现清空格式化的即插即用还原。
- **文件跨界融合 (Drag & Drop)**：
  - 新增事件劫持钩子，对拖放入境文件进行前置解析过滤。允许文本形 `.txt` 瞬发化为内部作品并入库；能够捕捉图像碎片转换 Base64 进行模态挂靠展示，无需繁琐交互。

## 5. 表现层逻辑校验与进阶UI机制
- **记忆与非阻塞型设置层**：
  - 去除旧时代警示窗，基于内部状态机接管防呆倒计时的安全确认模态层。
  - 高级排版设置中“基础排版”和“进阶排版”两个重要模块均挂载折叠器，相关缩放形态跟随 LocalStorage 长久化记录。
- **精准触发式动画池**：打字机入场效果从泛滥的刷新和跳转触发中摘解出来。应用内维护一个常驻内存 `Set` 集合追踪访问过的 ID，从逻辑层锁定：让打字机动画仅服务于每本书籍在单次用户进程内的“初次会客”。
- **工程化构建集**：保留自动化批处理流，包含 `npm run generate-keys` 内建驱动并向 `new_keys_export.txt` 输出密钥凭据；针对上报流预设极客快照抓取(携带平台, OS, 开启皮肤值)辅助调试。
